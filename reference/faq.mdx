# FAQ

Frequently asked questions about the infrastructure.

## General Questions

### What is this infrastructure for?

This infrastructure supports Agentic AI solutions on Google Cloud Platform, including integrations with 11Labs and CrewAI.

### What technologies are used?

- Terraform for Infrastructure as Code
- Google Cloud Platform services
- 11Labs for voice AI
- CrewAI for agent orchestration
